# 项目结构

 |-- 项目根目录
     |-- configs
     |   |-- config.js //反向代理在这配
     |   |-- weexplus.json //weexplus 配置文件
     |-- dist  //输出目录
     |-- platforms // 原生项目
     |   |-- android
     |   |-- ios 
     |-- plugins //插件目录
     |-- src  //代码目录
         |-- entry.vue //入口文件
         |-- router.js //路由
         |-- assets //资源文件，该文件夹下都可以荣root:访问，例如（root:file/index.html）
         |   |-- file //其它资源类目录
         |   |   |-- index.html
         |   |-- font //字体目录
         |   |   |-- font.ttf
         |   |-- img //图片目录
         |       |-- back.png
         |-- mixin  
         |   |-- appBoard.js
         |   |-- main.js //全局注册对象在这
         |-- views //界面文件
             |-- index.vue
             |-- component //自带组件目录
             |   |-- button.vue
             |   |-- head.vue
             |   |-- page.vue
             |   |-- tabHost.vue
             |   |-- tabitem.vue
             |   |-- wraper.vue
             |-- tab
                 |-- tab1.vue
                 |-- tab2.vue
                 |-- tab3.vue

# weexplus.json

以下罗列了可以改动的配置项，其余项未列出，请勿改动
修改此文件需要重新打原生包配置才会生效

{
  "debug":true,//是否是debug模式，true会展示设置刷新的按钮
  "showerror":true,// 是否显示js错误页面
  "isPortrait":true,// 首页是否是竖屏
  "jsVersion":2,//js版本，用于控制热更新版本
  "debugEntry":"http://192.168.0.102:8890/js/index.js",//模拟器加载的首页地址（模拟器无法扫码，故设此项）
}
