# 环境搭建

这里为详细的环境搭建过程，主要包含前端环境和原生环境
原生环境配置较为复杂，如果暂时不需要原生的包，可以先不管原生的环境

# 自动搭建

安装node.js。

安装nodejs 官网的安装包。

依次执行以下命令,然后按照提示操作

npm install weexplus -g
weexplus env

最终跑完看到以下界面 mac:

win： 如果自动安装报错，请手动搭建

# 手动搭建

如果自动搭建不成功才需要手动搭建

# 前端环境

安装node.js。

最简单的安装方法就是直接安装官网的安装包，至于其它的使用包管理或者源码安装的方法请自行搜索。

安装weex基础环境。

npm install weex-toolkit -g
npm install weex-builder -g

安装weexplus-cli

npm install weexplus -g

# 原生环境

如果暂时不打包可以先不配置原生环境，可以暂时下载开发包调试看效果

# android环境搭建

  1. 安装jdk 下载地址
  2. 下载并安装android studio
  3. 配置java环境变量 windows 配置 mac 配置
  4. 配置adb环境变量 windows 配置adb mac 配置adb

# ios环境搭建

ios相对简单

  1. 下载XCode,商店下载安装即可
  2. 安装Cocopod 打开控制台输入
pod search
