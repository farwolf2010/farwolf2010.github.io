# 快速开始

本章节旨在让大家快速体验weexplus的魅力，所以复杂的步骤我们都丢到后面再说

# 安装基础包

npm install weex-toolkit -g
npm install weexplus -g

# 创建项目

weexplus create

如下图

控制台cd进入新建的项目目录执行

npm install

安装依赖，不报错的话，项目的前端环境就好了

# 启动项目

接下来开启编译执行

npm run dev

在弹出的网页中f12打开开发者模式，将设备选择为iphone6/7/8,然后刷新浏览器即可
在右侧控制台中会出现一个二维码，这个二维码用于手机端开发使用
路由跳转后需要刷新浏览器才会刷新二维码
操作视频如下

# 手机看效果

下载开发app

安装完毕打开app，点设置->扫描二维码，扫描网页console中的二维码

# 无法看到效果？

  • 确保手机和电脑处于一个局域网
  • 确保电脑的防火墙打开
  • 安装过虚拟机的电脑确保虚拟网卡是否禁用
